## Description
***Below is the list of arguments taken by the shell***

| Argument | Description | Use |
---  |  --- | --- |
 cd | Move to the directory | `> cd C:\Users` |
 help | List the functions | `> help <function_name>` |
 exit | Exit the shell | `> exit` |
